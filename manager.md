## **Manager（管理器）**

管理器是一种特殊类型的容器。在软件中，你可以通过一个容器右侧是否有"**+**"图标来判断它是否是管理器——当然，大多数情况下确实如此，不过有些时候它们只是增强型容器。你最常遇到的管理器包括：**模块管理器、序列管理器、图层管理器和自动化关键帧管理器**。

操作管理器时，你可以使用一些特定的函数和属性来添加项目、删除项目或获取其所有项目的数组。

| 方法                    | 描述                                                                                                                                                                                                                                                                                                                                                                                                        | 示例                                                                                                                  |
| ----------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------- |
| `addItem([type])`       | 向管理器添加一个项目并返回它。<br/>有些管理器（如模块管理器或图层管理器）可以创建多种类型的项目，因此需要在 `type` 参数中指定要创建的类型。<br/>其他管理器（如序列管理器）不需要此参数，因为只有一种类型的序列。在这些情况下，调用函数时无需提供参数。                                                                                                                            | `var newOSCModule = root.modules.addItem("OSC");`<br/>`var newSequence = root.sequences.addItem();`                  |
| `removeItem(item)`      | 移除一个项目。`item` 可以是项目对象本身，也可以是表示其名称的字符串（会精确匹配 niceName、shortName 及其小写版本）。                                                                                                                                                                                                                                                                                          | `root.modules.removeItem(myModule);`<br/>`root.modules.removeItem("My Module");`                                     |
| `removeAll()`           | 移除所有项目。一次性处理所有项目，效率更高，也支持撤销/重做。                                                                                                                                                                                                                                                                                                                                               | `root.modules.removeAll();`                                                                                           |
| `getItems()`            | 返回该管理器中所有项目的数组。                                                                                                                                                                                                                                                                                                                                                                             | `var items = root.sequences.getItems();`<br/>`script.log("Num sequences : " + items.length);`                        |
| `getItemWithName(name)` | 根据名称查找并返回第一个匹配的项目。会精确匹配 niceName、shortName 及其小写版本。                                                                                                                                                                                                                                                                                                                            | `var seq = root.sequences.getItemWithName("My Sequence");`                                                           |
| `getItemAt(index)`      | 返回指定索引位置的项目。                                                                                                                                                                                                                                                                                                                                                                                   | `var curSequence = root.sequences.getItemAt(1);` // 0 是第一个，所以这会获取列表中的第二个序列                          |
| `getItemIndex(item)`    | 返回指定项目的索引。                                                                                                                                                                                                                                                                                                                                                                                       | `var index = root.sequences.getItemIndex(curSequence);`                                                               |
| `getItemBefore(item)`   | 返回指定项目之前的项目。                                                                                                                                                                                                                                                                                                                                                                                   | `var prevSequence = root.sequences.getItemBefore(curSequence);`                                                       |
| `getItemAfter(item)`    | 返回指定项目之后的项目。                                                                                                                                                                                                                                                                                                                                                                                   | `var nextSequence = root.sequences.getItemAfter(curSequence);`                                                        |
| `reorderItems()`        | 重新排序该管理器中的所有项目。                                                                                                                                                                                                                                                                                                                                                                             | `var mappingLayer = root.sequences.sequence.layers.mapping;`<br/>`mappingLayer.automation.reorderItems();`           |
